Tic Tac Toe Game

A Java GUI implementation of Tic-Tac-Toe built for the B202 Advanced Programming module at GISMA University of Applied Sciences.

**A. Features**

- Java Swing GUI
- Player vs Computer
- Shape selection: X, O, ▲, ■
- Win & draw detection
- Encapsulated board logic
- Simple AI move selection

B. Technologies Used

| Library         | Purpose                                      |
|----------------|----------------------------------------------|
| `tkinter`       | GUI creation and user interaction            |
| `filedialog`    | File selection dialog                        |
| `messagebox`    | Popup alerts and error messages              |
| `fitz` (PyMuPDF)| PDF parsing and text extraction              |
| `python-docx`   | DOCX file creation and formatting            |
| `threading`     | Background processing for smooth UI          |

C. Installation

1. Clone the repository git clone https://github.com/Chippo90/Advanced-Programming.git
2. Open in a IntelliJ IDEA Ultimate
3. Compile and run Main.java
   
D. Usage
1. Launch the app.
2. Click Browse PDF to select a file.
3. Click Convert to DOCX to begin conversion.
4. View progress and final word/page count.
5. Find the converted DOCX saved in the same directory.
